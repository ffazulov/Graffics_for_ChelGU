Файл "MadeGIF" при запуске создает гиф изображение "gif.gif".
Файл "ProgramGrafics" запускает прилолжение на фреймворке PyQT.
Файл "modules" является модулем для файла "ProgramGrafics", в котором используются функции для строения графика.
Папка "lenta_news_2020_cleared", будет создана после запуска файла "MadeGIF", имеет преобразованные файлы со статьями, для более быстрой работы с графиком.
